# VIM Config（Vim 配置）

## 一、插件简介
```
cscope：一款开源免费的 C/C++浏览工具，自带一个基于文本的用户界面，通过cscope可以很方便地找到某个函数或变量的定义位置、被调用的位置等信息。Cscope对 C /C++支持较好，也可以自己定制来支持Java和Perl、Python等脚本语言。Vim和gvim都提供了cscope接口，通过适当的配置，可以在Unix/Linux下实现变量、函数、文件等之间跳转，就像Windows下的Source Insight一样灵活易用。由于cscope是开源免费的，而且配合vim可以脱离鼠标，实现全键盘操作，方便快捷地浏览源代码，深受程序员、Geek等人士的喜爱。

ctags：扫描指定的源文件，找出其中所包含的语法元素，并将找到的相关内容记录下来。

代码浏览：
----------------------------------------------------
javabrowser.vim ：java代码树浏览工具，需要借助于ctags（http://ctags.sourceforge.net）
taglist.vim  http://vim-taglist.sourceforge.net/  ：支持各种语言的函数列表（代码树），同样需要借助ctags
---------------------------------------------------

自动完成：

---------------------------------------------------
 ```
